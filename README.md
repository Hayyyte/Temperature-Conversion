# Temperature Converter

A simple Python program using Tkinter to convert temperatures between Celsius and Fahrenheit.

## Description

This program allows users to input a temperature value in either Celsius or Fahrenheit and converts it to the corresponding temperature unit. Users can toggle between Celsius and Fahrenheit units with a button click.

## Features

- Convert temperatures between Celsius and Fahrenheit
- Toggle between Celsius and Fahrenheit units
- Clear and user-friendly graphical interface

## Installation

1. Make sure you have Python installed on your system.
2. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/hayyyte/temperature-converter.git
    ```
3. Navigate to the project directory:
   ```bash
    cd temperature-converter
   ```
4. Run the program:
   ```bash
    python temperature_converter.py
   ```

## Usage

1. Enter the temperature value in the provided input field.
2. Click the "Convert" button to see the converted temperature.
3. Toggle between Celsius and Fahrenheit units using the "Toggle Unit" button.
